# Проект: Геометрические фигуры

## Описание
Программа для вычисления параметров геометрических фигур.

## Участники
- Студент 1: Прямоугольник
- Студент 2: Треугольник

## Функционал
- Вычисление периметра прямоугольника/треугольника
- Вычисление площади прямоугольника/треугольника
- Вычисление диагонали прямоугольника/равнобедренность треугольника
